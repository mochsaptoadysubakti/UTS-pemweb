Pengembangan CSS

Lie memperkenalkan konsep Cascading Style Sheets pada 10 Oktober 1994. Pada saat itu, HTML digunakan terutama untuk mengatur konten, dan desain visual halaman web diatur melalui pengaturan inline yang membatasi fleksibilitas desain dan pemeliharaan. Lie ingin menciptakan sistem di mana desainer web dapat memiliki kontrol penuh atas tampilan sebuah situs tanpa harus mengubah struktur konten HTML-nya.

CSS memperkenalkan konsep "cascading", di mana gaya dapat diwariskan dari elemen induk ke elemen anak, memungkinkan fleksibilitas dalam penataan halaman. Hal ini memungkinkan pengembang untuk membuat desain yang lebih konsisten dan mudah dikelola. Dengan CSS, desainer dapat menentukan tata letak, warna, font, margin, dan elemen-elemen desain lainnya yang memisahkan tampilan dari konten.

Pada 1996, versi pertama CSS dirilis sebagai standar oleh World Wide Web Consortium (W3C), yang dipimpin oleh Tim Berners-Lee. Ini adalah tonggak penting dalam pengembangan web, karena memperkenalkan cara baru dalam merancang halaman web secara efisien.

Lie juga menjadi salah satu advokat paling vokal untuk penggunaan standar web yang terbuka, yang memungkinkan interoperabilitas antar peramban dan platform. Ia menulis buku berjudul "Cascading Style Sheets: Designing for the Web" bersama Bert Bos, yang juga berkontribusi pada pengembangan CSS. Buku ini menjadi salah satu referensi utama bagi para pengembang web dan desainer yang ingin mempelajari CSS lebih lanjut.
