H책kon Wium Lie adalah penemu bahasa CSS (Cascading Style Sheets), yang merupakan bahasa utama untuk menentukan tampilan dan tata letak halaman web. Ia mengembangkan konsep CSS pada tahun 1994 ketika bekerja di CERN, pusat penelitian tempat Tim Berners-Lee menciptakan World Wide Web. CSS adalah salah satu pilar utama pengembangan web, bersama HTML dan JavaScript, yang memungkinkan pemisahan antara konten dan presentasi visual.

Latar Belakang Pendidikan dan Karier

H책kon Wium Lie lahir di Norwegia pada 1965. Ia memiliki latar belakang yang kuat dalam ilmu komputer dan desain interaksi. Lie memperoleh gelar master di bidang Ilmu Komputer dari Massachusetts Institute of Technology (MIT) pada 1991. Setelah itu, ia bekerja di berbagai institusi teknologi dan penelitian.

Pengembangan CSS

Lie memperkenalkan konsep Cascading Style Sheets pada 10 Oktober 1994. Pada saat itu, HTML digunakan terutama untuk mengatur konten, dan desain visual halaman web diatur melalui pengaturan inline yang membatasi fleksibilitas desain dan pemeliharaan. Lie ingin menciptakan sistem di mana desainer web dapat memiliki kontrol penuh atas tampilan sebuah situs tanpa harus mengubah struktur konten HTML-nya.

Konsep "Cascading" pada CSS

CSS memperkenalkan konsep "cascading", di mana gaya dapat diwariskan dari elemen induk ke elemen anak, memungkinkan fleksibilitas dalam penataan halaman. Dengan CSS, desainer dapat menentukan tata letak, warna, font, margin, dan elemen-elemen desain lainnya yang memisahkan tampilan dari konten.

Pada 1996, versi pertama CSS dirilis sebagai standar oleh World Wide Web Consortium (W3C), yang dipimpin oleh Tim Berners-Lee. Ini adalah tonggak penting dalam pengembangan web, karena memperkenalkan cara baru dalam merancang halaman web secara efisien.

Karier Lanjutan dan Pengaruh

Setelah mengembangkan CSS, Lie terus bekerja dalam pengembangan standar web. Ia kemudian bergabung dengan Opera Software sebagai Chief Technology Officer (CTO), di mana ia memainkan peran penting dalam pengembangan browser web yang mendukung standar web terbuka.

Pengaruh Terhadap Standar Web

Lie menjadi salah satu advokat paling vokal untuk penggunaan standar web yang terbuka, yang memungkinkan interoperabilitas antar peramban dan platform. Ia menulis buku berjudul "Cascading Style Sheets: Designing for the Web" bersama Bert Bos, yang juga berkontribusi pada pengembangan CSS. Buku ini menjadi referensi utama bagi para pengembang web dan desainer yang ingin mempelajari CSS lebih lanjut.

Penghargaan dan Warisan

CSS telah menjadi standar yang sangat penting dalam dunia pengembangan web. Berkat kontribusi H책kon Wium Lie, situs web modern dapat memiliki tampilan yang menarik, responsif, dan konsisten di berbagai perangkat dan ukuran layar. Pada 2016, Lie menerima penghargaan dari Association for Computing Machinery (ACM) untuk kontribusinya dalam pengembangan web melalui CSS.
